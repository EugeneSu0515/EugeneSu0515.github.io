[macos]好物系列-鍵盤&滑鼠設定

發表於 2022-10-07 更新於 2023-03-01 分類於 macos 閱讀次數：
1.2k 2 分鐘

前情提要

筆者自從四年前開始買第一把機械式鍵盤後，深深著迷，雖然第一把鍵盤為茶軸，打過Cherry原廠鍵盤G80-3494後，完全回不去了，覺得直上直下超好打字，瘋狂買各家廠牌的紅軸。因為筆者吃飯用開發程式語言C#在mac上開發還是不友善，Mac還是淪為娛樂機，但自從dotnet core推出後，完全就是造福mac使用者啊，就會比較多時間在mac上面做研究及開發。

內容

Mac筆電上貼那種黏貼式筆電架，背包裡面放一把65%機械式鍵盤+外接觸控板及滑鼠，完全就是電腦宅宅的標準配備，這個狀態下，一般鍵盤都是Windows配置，除非買Mac專用鍵盤，鍵位才會與Mac筆電相符，主要差別在於一般Windows鍵盤接到Mac筆電上，預設行為為Windows鍵為對應成Mac上的Command鍵，Alt鍵對應成Mac上的Option鍵。

Windows系統配置 Ctrl Windows Alt
Mac OS配置 Control Command Option

以上對應配置就知道是除非你要隨時切換身體記憶，對於筆者來說，買不同牌子的機械式鍵盤，光不同牌子對應的快捷鍵就背不完了，因此對於Command跟Option位置對調這件事情無法接受。

鍵盤設定

主要是Mac鍵盤預設配置左command、左option換位置，使用外接鍵盤上完全是無敵，完全跟Mac筆電上的配置相符，因此使用上完全沒有gap，

https://karabiner-elements.pqrs.org/

筆者至少會換成以下配置(基本上這個軟體完全可以設定成某一個鍵換成完全不同的鍵)

Windows系統配置 Ctrl Windows Alt
Mac OS配置 Control Option Command

滑鼠設定

Apple公司一直都是以UX取勝，觸控板上的自然手勢是一大亮點，但若滑鼠接上使用，也持續套用「自然手勢」則就不那麼友善了。這個情境下，筆者想要的理想解決方式為觸控板維持「自然手勢」，滑鼠則套用非自然手勢，滾輪依照想要往下則往同個方向(下)滾的方式完成，救星出現了，就是今天要介紹的主角:

https://github.com/ther0n/UnnaturalScrollWheels

安裝方式則非常簡單，透過howbrew即可

接著設定權限即可，於Security & Privacy > Accessibility中找到UnnaturalScrollWheels打勾即可

結論

藉由這兩套，Mac筆電接上市面上的多數機械式鍵盤完全Work，更有彈性地使用鍵盤，可以同時兼顧到觸控板及滑鼠不同操作，簡直完美，介紹給朋友都說棒，希望這則文章有幫助到大家。

參考

  • https://github.com/ther0n/UnnaturalScrollWheels
